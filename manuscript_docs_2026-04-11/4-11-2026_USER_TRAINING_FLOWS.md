# User Workflow & Training Guide: USC-PIS
**Date:** April 11, 2026
**Target:** Clinic Staff & Student Body

---

## 1. Clinical Staff Workflows (High-Level)

### 1.1 Nurse: Patient Intake & Vital Signs
1.  **Search:** Use the **Smart Search** on the Patients page to locate the student.
2.  **Create Record:** Click "Add Medical Record".
3.  **Vitals:** Enter Blood Pressure, Temperature, Pulse, and Respiratory Rate.
4.  **Save:** The record is now available for the Doctor to review and add diagnosis.

### 1.2 Doctor: Medical Certificate Approval
1.  **Notification:** Receive an in-app alert for "Pending Certificate Approval".
2.  **Review:** Open the certificate record in `MedicalCertificatesPage`.
3.  **Assess:** Use the **Assess Fitness** action.
4.  **Action:** Select "Fit" or "Not Fit" and provide the medical rationale.
5.  **Issue:** Approve the certificate, which then becomes available for the student to download as a PDF.

---

## 2. Student/Patient Journey

### 2.1 Account Onboarding
1.  **Registration:** Sign up using `@usc.edu.ph` email.
2.  **Verification:** Check email for 6-digit code and enter it on the verification screen.
3.  **Role Selection:** Faculty/Staff must select "Faculty" role if applicable.
4.  **Profile Setup:** Complete the multi-step form (Personal -> Health History -> Emergency Contacts).
5.  **Dashboard Access:** Once `completeSetup` is True, the full dashboard (Campaigns/Records) is unlocked.

### 2.2 Health Record Management
1.  **Insights:** Navigate to `health-insights` to see a timeline of all medical and dental visits.
2.  **Certificates:** Click "Request Certificate" to start a new application for a field trip or absence.

---

## 3. Administrative Workflows

### 3.1 Report Generation
1.  **Select Type:** Navigate to the Reports menu and choose (e.g., "Monthly Visit Trends").
2.  **Filter:** Set the date range (e.g., 2nd Semester, AY 2025-2026).
3.  **Export:** Select "Excel" for data analysis or "PDF" for official submission.

### 3.2 Campaign Publication
1.  **Create:** Open the Campaign Management tool.
2.  **Upload:** Attach the Banner and PubMat images.
3.  **Schedule:** Set the start and end dates.
4.  **Publish:** Change status to "Active" to push to all student dashboards.
